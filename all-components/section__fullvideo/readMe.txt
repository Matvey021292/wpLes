Форматы, которые нужно залить в папку /video/ - mp4, ogv, webm, png, jpg, jpeg, gif, mp3.
Не забываем зжимать видео, макс. размер до 15МБ. Можно использовать онлайн сервисы или программу FormatFactory. Если использовать связку видео-аудио, то с видео обязательно нужно удалить звук!!
В section__fullvideo.html прописаны названия файлов - event (event.mp4, event.png ...).
Путь к аудио прописан в section__fullvideo.js.
Используется плагин vide -> jquery.vide.js
