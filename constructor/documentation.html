Для работы с данным конструктором необходимо иметь последнюю версию NodeJS, скачать которую можно на официальном сайте NodeJS, так как используются новые функции ES7.

Если после обновления NodeJS у вас возникает ошибка при запуске скрипта, необходимо прописать в консоли npm rebuild node-sass --force.

Компоненты в левой колонке необходимо выстраивать в таком порядке, в котором хотите их видеть на сайте.

После выбора нужных компонентов необходимо остановить в консоли процесс start с помощью Ctrl + C и запустить новый процесс gulp, который соберет весь сайт в папку build. При любых изменениях файлов в папке src браузер автоматически перезагрузится.

Если вы забыли сразу добавить нужный компонент, то останавливаем процесс gulp, запускаем npm start и выбираем все нужные компоненты, а не только новые, так как при копировании компонентов папка src/components будет очищаться.

Удачи в ваших начинаниях!
